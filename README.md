## Description
Here is an example of a project where logging is implemented using Winston, and the log data is sent to Grafana Loki for monitoring.

## Require Packgae 
```bash
npm winston
npm winston-loki
```

## Installation
To install the dependencies, run the following command:

```bash
npm install
```

```bash
docker-compose up
```
## Running the app

```bash
$ npm run dev
```
